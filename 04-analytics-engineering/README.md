For the data used in this project, I utilized the Green Taxi dataset from 2020 and the Yellow Taxi dataset from January to June 2024. I did not fully follow the dataset requirements specified in the homework because the original dataset size was too large. Uploading the entire dataset to BigQuery would consume a significant amount of internet data, and the internet connection in my country (Indonesia) is not always stable, which would make the upload process very time-consuming. Therefore, I decided to use this subset of data so that the project could still be completed effectively.
